# eight

A toy JPEG encoder/decoder in Clojure, drawing with Swing via Seesaw.

## Usage

````
$ lein deps
$ lein repl
user=> (require 'eight.core)
user=> (ns eight.core)
eight.core=> (draw-big)
````
